<h1>How to Run the Project Locally?</h1>
<ol>
  <li>Clone the project from GitHub</li>
  <li>cd backend</li>
  <li>npm install</li>
  <li>npm start</li>
  <li>Now the backend is running at http://localhost:5000</li>
  <li>cd ../frontend</li>
  <li>npm install</li>
  <li>npm start</li>
  <li>Now the web application will start on your browser</li>
</ol>
